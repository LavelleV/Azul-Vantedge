# Azul Vantedge Body Map Assets

Place the `assets/body-map` folder into the root of the Azul Vantedge project.

Included files:

- `assets/body-map/full/front-full-body.png`
- `assets/body-map/full/back-full-body.png`
- `assets/body-map/regions/hip-front.png`
- `assets/body-map/regions/hip-glute-back.png`

These are intended as the first production-style anatomical assets for the app. The app should keep the existing SVG fallback if an image is missing.
